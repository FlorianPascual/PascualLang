How to Use

  • To process text, click the icon in any text input or use the keyboard shortcut Ctrl+Shift+Y (Command+Shift+Y on Mac).
  • Use /translate to [language] at the beginning of your text to translate it. Example: /translate to Spanish Hello.
  • Select a tone preset or define a custom tone for text processing.
